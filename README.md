## Organização das soluções

Cada resolução deverá conter:

* Link para o problema original;
* Descrição resumida da abordagem utilizada;
* Análise de complexidade de tempo e espaço;
* Código-fonte comentado;
* Observações importantes sobre a solução.

Exemplo:

```text
Resolucoes_de_Problemas/
└── Codeforces/
    └── Nome_ou_Numero_do_Problema/
        ├── README.md
        └── solucao.cpp
```
